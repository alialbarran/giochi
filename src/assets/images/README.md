# Assets - Images

Este directorio contiene todas las imágenes y sprites del juego.

## Estructura Sugerida

- `sprites/` - Sprites de personajes y objetos
- `backgrounds/` - Fondos de niveles
- `ui/` - Elementos de interfaz de usuario
- `icons/` - Iconos y símbolos

## Formato Recomendado

- PNG con transparencia para sprites
- JPG para fondos grandes
- SVG para elementos escalables

## Convenciones de Nombres

- Usar nombres descriptivos en minúsculas
- Separar palabras con guiones: `snake-head.png`
- Incluir dimensiones si es relevante: `bg-1920x1080.jpg`

## Placeholder

Actualmente usamos emojis y colores CSS. Los assets gráficos pueden agregarse aquí para reemplazar los placeholders.
