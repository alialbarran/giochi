# Assets - Sounds

Este directorio contiene efectos de sonido y música del juego.

## Estructura Sugerida

- `sfx/` - Efectos de sonido (comer, chocar, etc.)
- `music/` - Música de fondo para menús y niveles
- `voice/` - Clips de voz (opcional)

## Formato Recomendado

- MP3 para compatibilidad amplia
- OGG como alternativa
- WAV para efectos cortos y de alta calidad

## Volúmenes

- SFX: mantener entre -10dB y -6dB
- Música: mantener entre -20dB y -15dB
- Normalizar todos los archivos para consistencia
